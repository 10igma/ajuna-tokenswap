## 5. Git Hygiene

### .gitignore — Critical Issue

Current `.gitignore` only contains:
```
node_modules
```

This is severely insufficient. The following generated/binary directories are being committed:

| Directory | Size | Generated By | Should Be Ignored |
|-----------|------|-------------|-------------------|
| `artifacts/` | 8.3 MB | `hardhat compile` | ✅ Yes |
| `cache/` | 44 KB | `hardhat compile` | ✅ Yes |
| `typechain-types/` | 992 KB | `typechain` | ✅ Yes |
| `chain-specs/` | 3.2 MB | `generate_chain_spec.sh` | ✅ Yes |

### Recommended `.gitignore`

```gitignore
# Dependencies
node_modules/

# Hardhat
artifacts/
cache/
typechain-types/

# Chain specs (generated)
chain-specs/

# Environment
.env
.env.local

# Ignition deployment artifacts
ignition/deployments/

# IDE
.vscode/
.idea/

# OS
.DS_Store
Thumbs.db

# Coverage
coverage/
coverage.json

# polkadot-sdk (external build, if not a submodule)
# polkadot-sdk/
```

### Impact

Removing tracked generated files after updating `.gitignore`:
```bash
git rm -r --cached artifacts/ cache/ typechain-types/ chain-specs/
git add .gitignore
git commit -m "chore: gitignore generated files"
```
